# 叔本华《人生的智慧》核心体系

提取自叔本华1850年晚年著作《人生的智慧》（原载《附录与补遗》），上海人民出版社2005年版，276页。

## 核心理念

> 幸福取决于**人的自身**（品格、智力、性情、健康），而非财产或外在表象。

## 人生幸福的三大分类

| 分类 | 内容 | 重要性 |
|:----:|:-----|:------:|
| ① 人的自身 | 健康、力量、外貌、气质、道德品格、精神智力 | 最高 |
| ② 人所拥有的财产 | 财富、占有物 | 中等 |
| ③ 人所展现的表象 | 名誉、地位、名声 | 最低 |

## 人的自身 — 幸福的内在源泉

1. **愉快的心情** — "它是幸福的现金，其他都是兑现幸福的支票。"
2. **健康** — 愉快心情从健康里长出的花朵。
3. **智力** — 思想丰富的人能把平凡之事化为趣味盎然。
4. **道德品格** — 高贵的品格是内在不可剥夺的财富。

## 财富观

- 财富是抵御不幸的城墙，不是寻欢作乐的许可证
- "财富犹如海水：一个人海水喝得越多，他就越感到口渴。"
- 金钱能满足抽象中的普遍需要，而其他物品只能满足单一需要

## 对待他人看法的态度

- 自身价值在于"你是什么"，而非别人认为你是什么
- 大多数人的头脑里是肤浅的思想和渺小的念头
- 不要抬举别人的轻蔑

## 实用格言（精选）

- 认识自己：知道自己应该从事何种职业、扮演何种角色
- 活在现在：只有现在才是唯一真实和确切的
- 局限即幸福：所有局限和节制都有助于增进幸福
- 独处的价值：我的拥有就在我身
- 控制想象力：想象会夸大不幸，用理性和判断力去观察
- 思想抽屉：把不同的事情放进不同的"抽屉"
- 放弃和忍受：给愿望规定限度，节制欲望
- 生命在于行动：创作、制作、学习——对于幸福绝对必要
- 保护大脑：大脑需要休息，睡眠是必需的
- 保持距离：社会人群如一堆火——太近会灼伤，太远会冷
- 少说话：与其让别人期待你说话，不如让别人惊讶你沉默
- 不争论：与无知的人争论只会拉低你的水平
- 要么孤独，要么庸俗

## 核心公式

```
幸福 = 健康 × (自身内在价值 / 欲望范围)
```

**三个不：**
- 不拿自身与他人比较（嫉妒）
- 不活在他人看法中（虚荣）
- 不为不可改变之事苦恼（悔恨）

**三个要：**
- 要保护健康（身体 + 大脑）
- 要发展自身（智力 + 品格）
- 要活在现在（此刻是唯一真实的）
